Este proyecto consiste en el desarrollo de una herramienta de análisis de datos llamada Explorador Pokemon. El programa, permite cargar un subconjunto de 50 registros del dataset original mediante el uso de bucles y la función open(), procesando la información línea por línea con el método split(). Para correr el programa, se debe asegurar que el archivo Pokemon.csv esté en la misma carpeta que el código main.py, abrir una terminal en dicha ubicación y ejecutar el comando python main.py. Una vez iniciado, el usuario encontrará un menú interactivo en consola para realizar búsquedas por término con conteo de resultados, visualizar estadísticas básicas de máximo, mínimo y promedio para atributos como el ataque, y aplicar filtros por condiciones específicas.
